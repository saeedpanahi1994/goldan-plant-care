# 📱 راهنمای آپدیت و دریافت نسخه جدید اپلیکیشن گلدان

تبریک بابت موفقیت در ساخت اپلیکیشن! 🎉
این فایل راهنمای قدم‌به‌قدم برای اعمال تغییرات و دریافت نسخه جدید APK است.

## 🔄 روند کلی
1. **تغییر کد**: تغییرات مورد نظر را در پروژه اعمال کنید.
2. **ارسال به گیت‌هاب**: کد را Push کنید.
3. **ساخت خودکار**: گیت‌هاب اکشنز به صورت خودکار APK جدید را می‌سازد.
4. **دانلود**: فایل APK را دانلود و نصب کنید.

---

## 📝 جزئیات مراحل

### ۱. اعمال تغییرات (در VS Code)
هر تغییری که می‌خواهید در ظاهر یا عملکرد اپلیکیشن بدهید، در پوشه `frontend` انجام دهید.
مثلاً تغییر متن‌ها، رنگ‌ها یا اضافه کردن قابلیت جدید.

### ۲. ارسال تغییرات (Push)
بعد از ذخیره فایل‌ها، دستورات زیر را در ترمینال VS Code اجرا کنید تا کدها به سرور فرستاده شوند:

```powershell
# 1. همه تغییرات را انتخاب کنید
git add .

# 2. تغییرات را ذخیره کنید (پیام دلخواه خود را بنویسید)
git commit -m " تغییرات جدید: تست نهایی   "

# 3. ارسال به گیت‌هاب (این دستور روند ساخت APK را شروع می‌کند)
git push
```

### ۳. دریافت نسخه جدید (APK)
1. به صفحه **Actions** در گیت‌هاب پروژه بروید:
   👉 [لینک مستقیم به بخش Actions](https://github.com/saeedpanahi1994/goldan-plant-care/actions)

2. آخرین گزینه در لیست (که نام کامیت شما را دارد) را پیدا کنید.
   - 🟡 **زرد**: در حال ساخت (حدود ۳ دقیقه صبر کنید)
   - ✅ **سبز**: انجام شد

3. روی آیتم **سبز شده** کلیک کنید.

4. به پایین صفحه بخش **Artifacts** بروید.

5. روی فایل **`goldan-debug-apk`** کلیک کنید تا دانلود شود.

### ۴. نصب
- فایل دانلود شده به صورت `.zip` است. آن را Extract کنید.
- فایل `app-debug.apk` را به گوشی منتقل کرده و نصب کنید.

## 💡 نکات مهم
- اگر بیلد **قرمز (Fail)** شد، روی آن کلیک کنید تا خطا را ببینید (معمولاً مشکل در کد React است).
- برای تغییر نسخه (Version)، فایل `frontend/package.json` را باز کنید و عدد `version` را بالا ببرید (مثلاً `0.1.0` به `0.1.1`).

مثلا

git add .; git commit -m "feat: add offline mode"; git push
